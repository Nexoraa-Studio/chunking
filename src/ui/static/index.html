Pipeline overview

Raw SOP PDF → docling (layout-aware extraction, no OCR) → structural chunking (group by heading, numbering-inferred levels, drop TOC tables) → coarse DP (exact dynamic-programming partition) → embed + evaluate + visualize.

Pure Python. No LangChain. CPU only. Embedding model: all-MiniLM-L6-v2 (384-D, L2 normalized).

Coarse DP — the chunking method we're shipping

For an ordered sequence of n structural units, find the globally-optimal contiguous partition minimizing:

cost(S) = (1 − mean_cos_sim_within(S)) + α · length_penalty(S) + β · heading_crossings(S)

Solved via O(n²) DP. Auto-calibrates target_k = round(total_chars / 2000) clamped to [12, 30]. Length penalty centered on target_chars = total_chars / target_k. Runs in ~2 ms on n=54.

Result on this SOP: 21 chunks, mean ≈1933c, R@5 ≈ 0.87 on synthetic heading queries.
